Проект "Какая ты булочка?"
- пользователю представлено 5 вопросов. Ему необходимо выбрать только один из вариантов ответа
- программа с графическим интерфейсом
- формат файла exe, подходит для операционных систем Windows.
Используемые модули:
- tkinter
- requests

Переменные:
label_1 - метка 1-го вопроса
label_2 - метка 2-го вопроса
label_3 - метка 3-го вопроса
label_4 - метка 4-го вопроса
label_5 - метка 5-го вопроса

button_send - кнопка для обработки результатов ответов пользователя
pic_croissant - изображение круассана
pic_donut - изображение пончика
pic_cracker - изображение сухаря
progress_bar - прогресс бар загрузки
